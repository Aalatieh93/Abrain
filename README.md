# Ahmad Brain - Gemini AI Chat

صفحة محادثة ذكية باستخدام Google Gemini AI

## المتطلبات

- Node.js مثبت على جهازك

## التثبيت والتشغيل

### 1. استنسخ أو حمل الملفات

### 2. ثبت المكتبات
```bash
npm install
```

### 3. شغّل الخادم
```bash
npm start
```

### 4. افتح المتصفح
اذهب إلى: `http://localhost:3000`

## الملفات

- `server.js` - خادم Express
- `public/index.html` - واجهة المستخدم
- `package.json` - المكتبات المطلوبة

## الميزات

✅ محادثة مباشرة مع Gemini AI
✅ واجهة سهلة وجميلة
✅ يدعم العربية والإنجليزية
✅ يعمل على الهاتف والحاسوب

## الدعم

إذا واجهت أي مشكلة، تأكد من:
- تثبيت Node.js بشكل صحيح
- تشغيل `npm install` أولاً
- فتح الرابط الصحيح في المتصفح
